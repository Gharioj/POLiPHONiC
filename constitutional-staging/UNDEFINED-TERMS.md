# Undefined Terms

An index of constitutional terms that are *named* somewhere in staged material but have no Definition, Principle, or Observation of their own anywhere in this repository — neither in `constitutional-staging/` nor in `constitutional-foundation/`. This file does not define them. It exists so a reviewer can find every such gap in one place rather than noticing them one at a time.

A term is removed from this list only when a definition for it is staged or integrated — never by writing one here.
