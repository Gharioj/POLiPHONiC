# About this site

This website is the official public documentation portal for **POLiPHONiC**. It is generated from, and reads directly from, the [`POLiPHONiC`](https://github.com/{{ site.repository }}) GitHub repository, which remains the authoritative source for every document presented here.

## Repository structure

- **`constitution/`** — the authoritative, operative constitutional text. Currently Working Draft 1.
- **`constitutional-foundation/`** — the constitutional research and discovery record: 39 integrated source documents, preserved unmodified, that the operative draft was assembled from and departs from. Presented on this site as [Framework]({{ '/framework/' | relative_url }}).
- **`constitutional-discoveries/`** — constitutional questions, tensions and anomalies noticed as they emerged, recorded in chronological order.
- **`constitutional-resolutions/`** — constitutional rulings recorded directly, linked to the Foundation without modifying it.
- **`constitutional-staging/`** — constitutional knowledge awaiting integration, held with its provenance recorded, prior to review. Not yet published on this site: it is explicitly pre-review material, not the operative or integrated record.
- **`manifesto/`** and **`brand/`** — the manifesto PDFs and wordmark assets presented under [Manifesto]({{ '/manifesto/' | relative_url }}) and [Downloads]({{ '/downloads/' | relative_url }}).

## Document status

- **Framework:** 39 documents integrated, preserved unmodified.
- **Discoveries:** 5 open, 0 resolved.
- **Resolutions:** 1 recorded.

## A note on this website

This site's typography and colour palette are a visual identity applied for the documentation website. No official POLiPHONiC typography or colour specification was found in the repository at the time this site was built, so a considered substitute was chosen. The POLiPHONiC wordmark and mark, shown in the site header and listed under Downloads, were supplied as image assets and are used as provided.

Nothing in `constitution/`, `constitutional-foundation/`, `constitutional-discoveries/`, or `constitutional-resolutions/` is modified by the existence of this website — every original file remains exactly as integrated in the repository.
